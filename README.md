# Packard Stopwatch

PWA секундомер - работает на любом устройстве прямо в браузере.

## Описание

Простой и функциональный секундомер с поддержкой:
- **Тап** - старт/пауза
- **Двойной тап** - сброс
- **PWA** - устанавливается на главный экран как приложение
- **Офлайн режим** - работает без интернета
- **Полноэкранный режим** - без адресной строки

## Использование

### Веб-версия (онлайн)

Откройте ссылку: **https://stexiel.github.io/PackardStopwatch**

### Установка на iPhone/Android

1. Откройте ссылку в Safari (iOS) или Chrome (Android)
2. Появится предложение "Установить приложение"
3. Нажмите **"Установить"**
4. Приложение появится на главном экране
5. Запускайте как обычное приложение

**Или вручную:**
- **iOS**: Открыть в Safari → Поделиться → На экран "Домой"
- **Android**: Открыть в Chrome → Меню → Установить приложение

## Разработка

Проект использует чистый HTML/CSS/JavaScript - не требует установки зависимостей.

### Локальный запуск

Откройте `web/index.html` в браузере или используйте любой локальный сервер:

```bash
cd web
python -m http.server 8000
```

Затем откройте http://localhost:8000

### Деплой

Автоматический деплой на GitHub Pages при каждом push в `main`.
