# Packard Stopwatch

Плавающий секундомер с жестами управления для iOS.

## Возможности

- ⏱️ Точный секундомер (минуты:секунды:миллисекунды)
- 🎨 Цветовая индикация (зеленый = работает, красный = пауза)
- 👆 Одиночный тап - старт/пауза
- 👆👆 Двойной тап - сброс
- 🖐️ Перетаскивание - переместить по экрану
- ⏳ Долгое нажатие - выход

## Сборка на Codemagic

1. Зарегистрируйтесь на https://codemagic.io
2. Подключите этот репозиторий
3. Запустите сборку
4. Скачайте .ipa файл из Artifacts

## Установка на iPhone

Используйте AltStore или Sideloadly для установки .ipa файла на iPhone.
