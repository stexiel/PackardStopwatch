# Packard Stopwatch - iOS Overlay Stopwatch

Секундомер для iOS с возможностью работы поверх других приложений.

## Функции

- ⏱️ **Секундомер**: Минуты:Секунды:Миллисекунды (формат MM:SS:MS)
- 🎨 **Цветовая индикация**:
  - Белый текст
  - Зеленый фон - когда работает
  - Красный фон - когда на паузе
- 👆 **Жесты управления**:
  - **Одиночное нажатие** - Старт/Пауза секундомера
  - **Двойное нажатие** - Сброс секундомера
  - **Перетаскивание** - Перемещение секундомера по экрану
  - **Длинное нажатие** - Выход из приложения

## Установка

1. Убедитесь, что Flutter установлен:
```bash
flutter --version
```

2. Установите зависимости:
```bash
flutter pub get
```

3. Запустите на iOS устройстве или симуляторе:
```bash
flutter run
```

## Сборка iOS приложения (.ipa)

**⚠️ ВАЖНО:** Для сборки iOS приложения требуется Mac с Xcode!

### Быстрый старт (на Mac):
```bash
flutter build ios --release
open ios/Runner.xcworkspace
```

Подробные инструкции смотрите в файле **BUILD_IOS.md**

## Требования

- Flutter SDK 3.0.0 или выше
- iOS 11.0 или выше
- Xcode (для сборки iOS приложения)

## Примечание

Для полноценной работы поверх других приложений на iOS требуется:
- Настройка Picture-in-Picture режима (требует дополнительной конфигурации)
- Или использование в качестве виджета на главном экране

iOS имеет ограничения на overlay приложения из-за политики безопасности Apple.
