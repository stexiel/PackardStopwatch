# Инструкция по сборке iOS приложения

## ✅ Что уже сделано:

1. ✅ Установлены npm зависимости
2. ✅ Установлен EAS CLI глобально
3. ✅ Проект загружен на GitHub

---

## 🚀 Следующие шаги (требуют вашего участия):

### Шаг 1: Логин в Expo

Откройте терминал в этой папке и выполните:

```bash
eas login
```

**Что делать:**
- Если у вас уже есть аккаунт Expo - введите email и пароль
- Если нет аккаунта - зарегистрируйтесь на https://expo.dev и вернитесь к этому шагу

---

### Шаг 2: Запуск сборки iOS

После успешного логина выполните:

```bash
eas build --platform ios --profile preview
```

**Что произойдет:**
1. EAS спросит несколько вопросов - везде нажимайте Enter (используем дефолтные настройки)
2. Проект загрузится в облако Expo
3. Начнется сборка iOS приложения (~15-20 минут)
4. Вы увидите ссылку на страницу сборки

---

### Шаг 3: Отслеживание сборки

Откройте ссылку которую покажет EAS CLI или перейдите на:

```
https://expo.dev
```

1. Войдите в свой аккаунт
2. Откройте проект **packard-stopwatch**
3. Перейдите в раздел **Builds**
4. Следите за прогрессом сборки

---

### Шаг 4: Скачивание .ipa

После успешной сборки (статус ✅ Success):

1. На странице сборки нажмите **Download**
2. Скачайте файл `.ipa`
3. Сохраните его на компьютер

---

### Шаг 5: Установка на iPhone

Используйте одну из программ:

**AltStore** (рекомендуется):
1. Скачайте с https://altstore.io
2. Установите на компьютер
3. Подключите iPhone через USB
4. Откройте AltStore и установите .ipa файл

**Sideloadly**:
1. Скачайте с https://sideloadly.io
2. Установите на компьютер
3. Подключите iPhone через USB
4. Перетащите .ipa в Sideloadly и нажмите Start

---

## 🎯 Готово!

После установки приложение появится на вашем iPhone как обычное приложение.

**Управление:**
- 👆 Tap - старт/пауза
- 👆👆 Double Tap - сброс
- 🟢 Зеленый = работает
- 🔴 Красный = пауза
