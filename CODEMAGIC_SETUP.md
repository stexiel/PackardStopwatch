## 🚀 Настройка Codemagic (ваши действия):

### Шаг 1: Регистрация на Codemagic

1. Откройте https://codemagic.io
2. Нажмите **"Sign up for free"**
3. Войдите через **GitHub**
4. Разрешите доступ к репозиториям

---

### Шаг 2: Добавление проекта

1. После входа нажмите **"Add application"**
2. Выберите репозиторий **PackardStopwatch**
3. Codemagic автоматически найдет файл `codemagic.yaml`
4. Нажмите **"Finish: Add application"**

---

### Шаг 3: Запуск сборки

1. На странице проекта нажмите **"Start new build"**
2. Выберите workflow: **"Expo iOS Build"**
3. Выберите ветку: **main**
4. Нажмите **"Start build"**

---

## ⏱️ Процесс сборки (~15-20 минут):

Codemagic выполнит следующие шаги:

1. ✅ Установка npm зависимостей
2. ✅ Генерация нативного iOS проекта через `expo prebuild`
3. ✅ Установка CocoaPods
4. ✅ Отключение code signing
5. ✅ Сборка iOS приложения через xcodebuild
6. ✅ Создание unsigned .ipa файла

**Следите за прогрессом на странице сборки.**

---

## 📥 Скачивание .ipa файла:

После успешной сборки (статус ✅ Success):

1. Прокрутите вниз до раздела **"Artifacts"**
2. Найдите **"PackardStopwatch-unsigned.ipa"**
3. Нажмите на файл чтобы скачать
4. Сохраните на компьютер

Также вы получите email с результатом сборки на **stexielgithub@gmail.com**

## 📱 Установка на iPhone:

### Вариант 1: AltStore (рекомендуется)

1. Скачайте AltStore: https://altstore.io
2. Установите на компьютер
3. Подключите iPhone через USB
4. Откройте AltStore
5. Перетащите .ipa файл в AltStore
6. Нажмите **"Install"**

### Вариант 2: Sideloadly

1. Скачайте Sideloadly: https://sideloadly.io
2. Установите на компьютер
3. Подключите iPhone через USB
4. Откройте Sideloadly
5. Перетащите .ipa в окно программы
6. Введите Apple ID
7. Нажмите **"Start"**
